# ACL

Access Control List is a feature that allows you to control the access to features on your server for specific players.

To use ACL, refer to the [PlayerClient](./playerclient.mdx) API.

### AclLevel

| Level | Description |
|-------|-------------|
| `AclLevel.Owner` | Highest access level, has full control over the server |
| `AclLevel.Admin` | High access level, can manage most server features |
| `AclLevel.Moderator` | Moderate access level, can manage basic server features |
| `AclLevel.Normal` | Default access level for all players |
