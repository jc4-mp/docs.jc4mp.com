# Weapon List

This is a list of all weapons available in JC4MP. You can give weapons to players using the [`NetPlayer:GiveWeapon`](/server-api/netplayer) function.

```lua
-- Give a spoecific player an assault rifle with 100 ammo, equipped, and in the primary weapon slot
player:GiveWeapon("wpn_000_assault_rifle", 100, true, WeaponSlot.Primary)
```
