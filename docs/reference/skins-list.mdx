# Skins List

This is a list of all skins available in JC4MP. You can set a player's skin using the [`Local.SetSkin`](/client-api/local) function.

```lua
-- Set a player's skin to the default Rico skin
Local.SetSkin("rico")

-- Set a player's skin to the Ghost Enemy skin
Local.SetSkin("ghost_enemy_001")
```
