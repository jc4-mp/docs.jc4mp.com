# NetRigidObject

This is the client-side class for NetRigidObject.

## Class Instance Methods

<NetRigidObjectShared />

#### `NetRigidObject:GetAABB(): AABB`

Returns the AABB (bounding box) of the rigid object.

**Returns:** [AABB](/shared-api/aabb) - The AABB of the rigid object. This is a table containing two fields: `min` and `max`, both of which are vec3.

**Example:**
```lua
local aabb = rigidObject:GetAABB()
print("Rigid object AABB: " .. tostring(aabb.min) .. " " .. tostring(aabb.max))
```

#### `NetRigidObject:GetLocalAABB(): AABB`

Returns the local AABB (bounding box) of the rigid object.

**Returns:** [AABB](/shared-api/aabb) - The local AABB of the rigid object. This is a table containing two fields: `min` and `max`, both of which are vec3.

**Example:**
```lua
local localAABB = rigidObject:GetLocalAABB()
print("Rigid object local AABB: " .. tostring(localAABB.min) .. " " .. tostring(localAABB.max))
```
