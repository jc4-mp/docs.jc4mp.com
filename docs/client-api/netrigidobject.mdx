#### `NetRigidObject:GetAABB(): AABB`

Returns the AABB (bounding box) of the rigid object.

**Returns:** [AABB](/shared-api/primitives/aabb) - The AABB of the rigid object. This is a table containing two fields: `min` and `max`, both of which are vec3.

**Example:**
```lua
local aabb = rigidObject:GetAABB()
print("Rigid object AABB: " .. tostring(aabb.min) .. " " .. tostring(aabb.max))
```

#### `NetRigidObject:GetLocalAABB(): AABB`

Returns the local AABB (bounding box) of the rigid object.

**Returns:** [AABB](/shared-api/primitives/aabb) - The local AABB of the rigid object. This is a table containing two fields: `min` and `max`, both of which are vec3.

**Example:**
```lua
local localAABB = rigidObject:GetLocalAABB()
print("Rigid object local AABB: " .. tostring(localAABB.min) .. " " .. tostring(localAABB.max))
```

#### `NetRigidObject:EnableCollision(enable: boolean): void`

Enables or disables collision for this net rigid object on the client.

**Parameters:**
- `enable: boolean` - Whether to enable (`true`) or disable (`false`) collision

**Example:**
```lua
rigidObject:EnableCollision(false)
-- ... later
rigidObject:EnableCollision(true)
```
