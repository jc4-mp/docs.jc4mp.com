# Players

These APIs allow you to interface with Players in the game. This inherits from the shared [Players](/shared-api/players) class.

## Global Methods

#### `Players.Local(): PlayerClient`

Returns the local [player](/client-api/playerclient). (this is the user's local player)


#### `Players.SetTagsEnabled(enabled: boolean)`

Enables or disables nametags / health bars for all players.


#### `Players.AreTagsEnabled(): boolean`

Returns whether nametags / health bars are enabled for all players.
