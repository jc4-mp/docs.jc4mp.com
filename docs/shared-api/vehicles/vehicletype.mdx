# VehicleType

The `VehicleType` enum is used to categorize vehicles.

| Vehicle Type | Description |
|---|---|
| `VehicleType.Motorbike` | Represents a motorbike |
| `VehicleType.Car` | Represents a car |
| `VehicleType.SeaVehicle` | Represents a sea vehicle |
| `VehicleType.Airplane` | Represents an airplane |
| `VehicleType.Helicopter` | Represents a helicopter |
| `VehicleType.Dirigible` | Represents a dirigible |
