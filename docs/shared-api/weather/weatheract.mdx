#  WeatherAct

WeatherAct defines different types of weather events that can be spawned in the world.

#### `WeatherAct.SpawnTornado (position: vec3, form_time: number)`

Args:
 - `position: vec3` - The position to spawn the tornado.
 - `form_time: number` - The time it takes to form the tornado in seconds.

See [Weather](/server-api/weather) for more information.

#### `WeatherAct.DespawnTornado (position: vec3, despawn_time: number)`

Args:
 - `position: vec3` - The position to despawn the tornado.
 - `despawn_time: number` - The time it takes to despawn the tornado in seconds.

See [Weather](/server-api/weather) for more information.
