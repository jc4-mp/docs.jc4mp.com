# TornadoState

TornadoState defines the state of a tornado.

See [Weather](/server-api/weather) for usage examples.

| State | Description |
|-------|-------------|
| Forming | The tornado is in the process of forming |
| Moving | The tornado is actively moving and fully formed |
| Stopping | The tornado is in the process of stopping |
| Stopped | The tornado has stopped moving |
| Dissipating | The tornado is in the process of dissipating |
| Disappearing | The tornado is disappearing |
| WaitingBeforeResetting | The tornado is waiting before resetting its state |
