# CameraShakeType

The `CameraShakeType` enum represents different types of camera shake effects.

## Values

| Value | Description |
|-------|-------------|
| `CameraShakeType.SpatialTranslation` | Spatial translation shake |
| `CameraShakeType.SpatialRotation` | Spatial rotation shake |
| `CameraShakeType.AbsoluteTranslation` | Absolute translation shake |
| `CameraShakeType.AbsoluteRotation` | Absolute rotation shake |
| `CameraShakeType.Disabled` | Disabled shake (no shake effect) |

## Usage Example

```lua
-- Spawn a spatial translation shake at a position
Camera.SpawnShake(CameraShakeType.SpatialTranslation, vec3(1000, 100, 2000), vec3(1.0, 1.0, 1.0), 5.0, 10.0, 2.0, 0.5)

-- Spawn an absolute rotation shake
Camera.SpawnShake(CameraShakeType.AbsoluteRotation, vec3(0, 0, 0), vec3(0.5, 0.5, 0.5), 3.0, 15.0, 1.5, 0.3)
```
