# POIDisplayMode

A POI display mode. See [POI](/client-api/poi) for more information.

| POIDisplayMode | Description |
|--------|-----------------------------|
| `POIDisplayMode.World`         | The POI only appears in the world |
| `POIDisplayMode.Map`            | The POI only appears on the map |
| `POIDisplayMode.WorldAndMap`       | The POI appears in both the world and the map |
