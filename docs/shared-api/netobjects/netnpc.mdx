# NetNPC

`NetNPC` shares the same base methods as [`NetObject`](/shared-api/netobjects/netobject) and adds only an identifier accessor.

#### `NetNPC:GetId(): string`

Returns the unique identifier of the NPC.

**Returns:** string - The unique identifier of the NPC

**Example:**
```lua
local id = npc:GetId()
print("NPC ID: " .. id)
```
