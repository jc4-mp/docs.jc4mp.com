#### `NetPlayer:GetWeaponAmmo(): number`

Returns the current ammo count for the equipped weapon module.

**Returns:** number - Current ammo for the active weapon module

#### `NetPlayer:GetInventoryAmmo(type: AmmoType): number`

Returns the current ammo count for the specified inventory ammo type.

**Parameters:**
- `type: AmmoType` - The ammo type to query. See [AmmoType](/shared-api/enums/ammotype).

**Returns:** number - Current ammo for the requested type
