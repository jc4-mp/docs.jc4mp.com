# WeaponSlot

`WeaponSlot` is global enum that represents the different weapon slots in JC4MP.

| WeaponSlot | Description |
| --- | --- |
| `WeaponSlot.Primary` | Primary weapon slot |
| `WeaponSlot.Secondary` | Secondary weapon slot |
| `WeaponSlot.Heavy` | Heavy weapon slot |
| `WeaponSlot.Max` | Max weapon slot |
| `WeaponSlot.None` | No weapon slot |
